# Kalika Lyceum website

Minimal static site for GitHub Pages with:
- Home page
- Privacy Policy
- Terms & Conditions
- Disclaimer

## Files
- `index.html`
- `privacy-policy.html`
- `terms.html`
- `disclaimer.html`
- `style.css`

## Custom domain
Add a file named `CNAME` containing:

kalikalyceum.com

Then configure the custom domain in GitHub Pages settings.
